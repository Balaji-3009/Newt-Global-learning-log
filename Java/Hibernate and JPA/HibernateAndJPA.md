# Hibernate and JPA

Platform: Udemy  
Duration: May 17 – May 18  


## May 17
- Started the course by exploring ORM (Object Relational Mapping) fundamentals and Hibernate architecture
- Understood JPA as a persistence specification and Hibernate as its implementation
- Reviewed entity mapping concepts and relationship handling between Java objects and database tables
- Explored Hibernate Session and persistence lifecycle basics
- Observed integration flow between Spring Boot and JPA repositories

## May 18
- Focused extensively on advanced JPA and Hibernate concepts used in enterprise backend applications
- Learned detailed entity relationship mappings including One-to-One, One-to-Many, and Many-to-Many associations
- Explored lazy loading and eager loading strategies and their impact on application performance
- Studied JPQL (Java Persistence Query Language) and advanced query execution workflows
- Learned pagination, sorting, and custom query handling using JPA repositories
- Understood transaction management and persistence context synchronization
- Explored inheritance mapping strategies and advanced entity lifecycle handling
- Studied caching concepts and Hibernate performance optimization basics
- Observed repository layer design and database abstraction techniques in Spring-based applications
