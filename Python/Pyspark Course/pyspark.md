# PYSPARK End to End Developer Course (Spark with Python)

Platform: Udemy  
Duration: May 11 – May 13  

---

## May 11
- Started the course by exploring Spark execution architecture and distributed processing concepts
- Understood Spark cluster components, executors, drivers, and task execution flow
- Reviewed SparkSession and advanced RDD concepts

- Explored practical usage of RDDs in distributed data processing workflows

- Learned advanced DataFrame operations including filtering, joins, aggregations, sorting, and window functions

- Understood Spark SQL architecture and query execution flow
- Studied performance optimization concepts including caching, repartitioning, coalesce, and execution planning
- Explored DAG scheduler, task scheduler, and narrow vs wide transformations in Spark execution

## May 13
- Learned about Spark shared variables and persistence techniques
- Explored HDFS integration and distributed storage concepts
- Revised Spark SQL, DataFrame APIs, and optimization concepts
- Observed practical big data processing workflows and ETL design approaches
